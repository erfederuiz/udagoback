Golang Simple CRM Backend.

Brief overview of the API

This simple backend system implements a CRUD API on top of a map type structure.

This map will store elements of custom type defined in the following structure:

type Customer struct { Id string `json:"id"` Name string `json:"name"` Role string `json:"role"` Email string `json:"email"` Phone int `json:"phone"` Contacted bool `json:"contacted"` }

A group of five elements will be created automatically on system strartup.




Getting all customers

Using a GET method on the following path the information about all customers will be retrieved.

/customers




Getting a single customer

Using a GET method on the following path the information about a customer will be retrieved.

The customer's Id must be provided as part of the URL

/customers/{id}




Creating a customer

Using a POST method on the following path the information about a customer will be update.

/customers

In addition a request body with the following format, must be provided:

{ "name": "Updated Name", "role": "Updated Role", "email": "Updated Email", "phone": integer, "contacted": boolean }




Updating a customer

Using a PATCH method on the following path the information about a customer will be update.

The customer's Id must be provided as part of the URL

/customers/{id}


In addition a request body with the following format, must be provided:

{ "name": "Updated Name", "role": "Updated Role", "email": "Updated Email", "phone": integer, "contacted": boolean }




Deleting a customer

Using a DELETE method on the following path the information about a customer will be deletd.

The customer's Id must be provided as part of the URL

/customers/{id}
